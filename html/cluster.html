Cluster layout is very similar to Tree diagrams except that all leaf nodes of the tree are placed at the same depth. Similarily, the Cluster layout have nodes that are connected by branches and shows relations between these nodes. The Cluster layout also has a root node, branches, child and leaf nodes.
