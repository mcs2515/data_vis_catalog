Spatial Position

The Partition Layout uses areas (arcs or rectangles) and placement rather than drawing links to each node to represent hierarchy. In a Partition Layout, nodes are dawn as solid areas and their placement relative to other nodes determines their position in the hierarchy. The size of each node represents a quantitative value that would be much hard to show in a tree diagram.
