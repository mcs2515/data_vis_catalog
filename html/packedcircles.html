Circle Packing

Marks:

Containment

Point

Channel:

Area

Circle Packing is similar to Treemap layout, but rather than drawing links to each node, Packed Circle layout uses circles instead to represent nodes in a hierarchal structure. Each circle contained in another circle represents a level in the hierarchy. The area and color of each circle can add more quantitative information about the node. The downside of the Packed Circle Layout is that it creates a lot of wasted space within the circles.
